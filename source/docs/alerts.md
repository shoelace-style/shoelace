## Alerts

Create an alert by applying the `alert` class to an element such as a `<div>`. You can change an alertâ€™s appearance using the `alert-*` modifier.

```html
<div class="alert">Default</div>
<div class="alert alert-success">Success</div>
<div class="alert alert-info">Info</span>
<div class="alert alert-warning">Warning</div>
<div class="alert alert-danger">Danger</div>
<div class="alert alert-inverse">Inverse</div>
```
